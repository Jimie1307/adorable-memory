How to make SSH key:

1) Type in terminal: ssh-keygen -t rsa -b 4096 -C "Github email"
2) Then, set the filename that will save the key.
3) Enter passphrase if you want. Can leave blank.
Result: Identification saved in the file. Public key saved in the file in .pub. Key fingerprint is added. Key's randomart image is provided.
Use private key to push files into Github.

Git Branching

1. All commits are under master branch.
2. A commit can have numbers of commit under it. It is called feature branch.
3. Each branch minds their own business. They dint know what other branches are doing.
BENEFITS: functions as a 'sandbox area' for updating a feature that may have broken code. So that does not affect the master branch.Can be connected to main branch if it is finished.
4. Hot Fix Branch: A SOS branch for quick bug fix.

How to branch


1. git branch: check what branch are you on and how many there are
2. git checkout -b [name-of-branch]: to make a new branch. to switch branches, type the same thing but without '-b'. 3. git diff: shows what changes and compares two versions of the code.
4. git merge master: merge the other branche with main branch
